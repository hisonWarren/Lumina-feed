# Changelog

本文件记录 Lumina Feed 各版本的用户可见变更。完整提交历史见 [GitHub Releases](https://github.com/hisonWarren/Lumina-feed/releases)。

## [0.4.95] — 2026-07-09

### 新增

- **设置 · 远程模型清单**：应用启动后从 GitHub 拉取 `config/model-catalog.json`（24h 缓存 + 内置兜底），无需发版即可增补 GPT-5.5 等新模型；设置页可手动刷新清单。
- **订阅简报 · 摘要恢复**：今日推送与历史简报重新展示文献摘要（可展开/折叠）；列表 IPC 自动补全缺失摘要。

### 改进

- **订阅简报 · Hero/Reader**：扫描视图仅展示简报条，完整报告在阅读器视图打开；单订阅/全部订阅模式标签更清晰。
- **检索页**：FindFetch 增加「非数据库分面」提示，避免与文献库筛选混淆。

### 维护

- 结构验证套件 63/63 全绿（阅读器、文献库、Shell 历史契约同步）。

## [0.4.94] — 2026-07-09

### 改进

- **设置 · 模型列表**：保存 API Key 后刷新可拉取账户可用对话模型全量列表（推荐项置顶），不再只显示 4 个精选；扩充各提供方内置推荐清单；过滤 embedding/语音等非对话模型。

### 修复

- **设置 · 模型下拉**：点击框外可关闭选项窗；下拉区增高可滚动。

## [0.4.93] — 2026-07-09

### 修复

- **设置 · 大模型提供方切换**：修复切到 Claude 等提供方后，模型下拉仍显示上一家的列表、底部仍残留上一次「连接成功」结果。切换时作废进行中的 `listModels` 请求并清空测试状态；成功提示标注当前提供方名称。

## [0.4.92] — 2026-07-08

### 新增

- **阅读器 · 助手追问记忆（L1+L2）**：追问时注入最近 2 轮问答的带页码要点，以及本篇已生成的总结/大纲摘录（仅助指代）；全文仍走 map-reduce 重新接地，结论须重新标注页码。UI 提示追问范围。

### 修复

- **我的文献 · 选择导出**：v0.4.91 误将导出绑定「跨篇分析」勾选；现改为标题栏独立「选择导出」按钮与专用勾选框，与跨篇分析完全分离。未开启时仍导出当前列表全部。

## [0.4.91] — 2026-07-08

### 新增

- **我的文献 · 导出所选**（已在 0.4.92 修正交互）：开启「跨篇分析」并勾选文献后导出所选——**请升级 0.4.92 使用正确的「选择导出」**。

## [0.4.90] — 2026-07-08

### 修复

- **我的文献 · 跨篇分析**：分析结果面板在滚动区外撑满视口导致无法滚动；现移入 `.lib-body` 与文献列表同区滚动。
- **阅读器 · 页内翻译**：长页单次 `maxTokens=1200` 导致译文截断遗漏；改为按段落分块翻译并动态提高 token 上限；略放宽 PDF 取文页眉页脚裁切。

### 新增

- **期刊 · WoS 详情**：逐刊/在线查询时拉取 wos-journal.info 详情页，展示 5 年 IF、WoS 核心收录（SCIE/SSCI 等）、学科类别、Best Ranking、OA 支持、WoS 状态、出版商等。

## [0.4.89] — 2026-07-06

### 修复

- **期刊 · 数据集面板布局**：展开「数据集 · 手动更新」后搜索栏被滚出视口、下方出现大片空白/底色。数据集面板改到搜索栏正下方；展开时滚回顶部保留搜索栏；展开时隐藏重复的空状态占位；期刊页容器补全背景色。

## [0.4.88] — 2026-07-05

### 新增

- **阅读器 · 译文侧栏右键菜单**：修复译文区选区右键无菜单（此前被阅读器/全局菜单双重排除）。选区菜单：复制、复制带页码引用、全选；段内对照模式下增加「复制中英对照」。空白处右键：复制本页译文、全选本页。

## [0.4.87] — 2026-07-05

### 修复

- **阅读器 · 多 PDF 标签页**：打开第二个及后续文档时，缩略图列表无法滚动、点击缩略图会跳回第 1 页。根因是页面/缩略图 DOM 使用全局 `id`，隐藏标签页中的节点被误选中；现改为在各自标签容器内按 `data-rd-page` / `data-rd-thumb` 查询，并避免异步恢复阅读位置时覆盖用户已跳转的页码。
- **检索取文 · 粘贴后回车**：在搜索框粘贴内容后直接按 Enter 不触发检索；现于 Enter 时读取输入框当前值再执行搜索。
- **期刊 · 粘贴后回车**：期刊检索框存在相同问题，已一并修复。
